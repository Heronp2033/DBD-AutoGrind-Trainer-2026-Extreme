## 💡 Troubleshooting
If the software does not work, try these steps:

- Run the trainer as an administrator. Many game tools require elevated permissions to read game memory.
- Install the latest version of the .NET Desktop Runtime. You can find this on the official Microsoft support page.
- Turn off your antivirus software for a moment to see if it blocks the injection process. Add an exception for the trainer folder if you find that it works after disabling the scanner.
- Restart your game client if the ESP or aimbot features do not show up during a match.
- Check that your display settings match the game settings. Fullscreen windowed mode often works better than exclusive fullscreen mode when using overlays.
